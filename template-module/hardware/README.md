# hardware

Place KiCad schematic and PCB files for this module in this directory.

Suggested naming convention:

```
hardware/
├── <module-name>.kicad_pro   # KiCad project file
├── <module-name>.kicad_sch   # Schematic
├── <module-name>.kicad_pcb   # PCB layout
└── fab/                      # Gerber / drill files for fabrication
```
